# data.gouv.fr — Portail données élections
Source : https://www.data.gouv.fr/elections
Éditeur : data.gouv.fr
Catégorie : elections

Les données relatives aux élections Cette rubrique rassemble les principaux jeux de données par type de scrutin : municipales, législatives, présidentielles, etc. Retrouvez l'ensemble des données électorales publiées par l'État et les organisations compétentes, pour vous permettre de : Analyser les tendances par type d'élection , pour vos projets, articles ou recherches Croiser les données électorales avec d'autres jeux de données territoriales Suggérer l'ouverture de données relatives aux élections Sélectionnez un scrutin pour accéder aux jeux de données disponibles. Données des élections municipales Cette page rassemble toutes les données relatives aux élections municipales : résultats par bureau de vote, informations sur les élus et toutes les données utiles permettant de comprendre les enjeux propres aux communes. Données des élections législatives Cette page rassemble l'ensemble des données relatives aux élections législatives : résultats par circonscription et données sur le fonctionnement et la transparence du pouvoir législatif, afin d'éclairer les enjeux de l'Assemblée nationale. Données des élections présidentielles Cette page permet d'accéder aux différentes données des élections présidentielles : résultats par tour, séries historiques, etc., ainsi que les données pertinentes pour analyser les dynamiques politiques à l'échelle nationale. Données des élections sénatoriales Cette page rassemble toutes les données relatives aux élections sénatoriales : résultats depuis 1992, comptes des campagnes, ainsi que toutes les données permettant d'éclairer les enjeux du Sénat. Autres données électorales et démocratiques Cette page rassemble les données des autres scrutins (élections sénatoriales, élections européennes...) des référendums et des consultations.
